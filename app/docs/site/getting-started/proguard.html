    ProGuard

    Proguard is supported using the standard "proguard-android.txt" configuration. The bundled consumer proguard file covers the proguard configuration for the SDK but you might have to do configurations for your application.

    In your apps build.gradle file:

    1
2
3
4
5
        release {
      minifyEnabled true
      shrinkResources true
      proguardFiles getDefaultProguardFile('proguard-android.txt'), "proguard.cfg"
    }

    In case you use logback-android-core, add this to your proguard.cfg file:

    -dontwarn javax.naming.InitialContext
