    Seos status words

    Background

    Seos uses status words defined in ISO 7816 documents, mainly ISO 7816-3 and ISO 7816-4. While the integrating application generally don't have to care about these errors, they are kept here for reference

    Error Meaning
    9000 No error
    6999 Applet selection failed
    61xx Response bytes remaining
    6E00 CLA value not supported
    6986 Command not allowed (no current EF)
    6985 Conditions of use not satisfied
    6C00 Correct Expected Length (Le)
    6984 Data invalid
    6988 Secure messaging data objects incorrect
    6A84 Not enough memory space in the file
    6983 File invalid
    6A82 File not found
    6A81 Function not supported
    6A86 Incorrect parameters (P1,P2)
    6D00 INS value not supported
    6A83 Record not found
    6982 Security condition not satisfied
    6F00 No precise diagnosis
    6A80 Wrong data
    6700 Wrong length
    6B00 Wrong parameters (P1,P2)
